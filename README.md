# 🚗 DVLD System (Driving License Department)

## 📌 Overview
The DVLD (Driving License Department) System is a desktop application designed to simulate real-world driving license management processes.  
It allows managing applicants, handling license requests, and processing approval workflows in a structured and efficient way.

This project demonstrates strong fundamentals in C#, OOP, and layered architecture, with a focus on building complete business workflows.

---

## ✨ Features
- Manage applicants and personal information  
- Apply for different types of driving licenses  
- Multi-step license processing workflow  
- Input validation and error handling  
- Data persistence using SQL Server  
- Structured and maintainable code using 3-tier architecture  

---

## 🏗️ Architecture
The application follows a 3-Tier Architecture:

- Presentation Layer  
  Windows Forms UI for user interaction  

- Business Logic Layer  
  Handles application rules, validation, and workflows  

- Data Access Layer  
  Manages database operations using ADO.NET  

This separation improves maintainability, scalability, and code organization.

---

## 🛠️ Technologies Used
- C#  
- Windows Forms  
- SQL Server  
- ADO.NET  
- Object-Oriented Programming (OOP)  

---

## 🚀 How to Run
1. Clone the repository  
2. Open the solution in Visual Studio  
3. Configure the SQL Server connection string  
4. Restore the database (if backup provided)  
5. Run the application
